# Controle de Estoques (Média Ponderada)

Este repositório contém uma planilha em Excel para **controle de estoques** utilizando o método da média ponderada.  
O objetivo é facilitar o acompanhamento de entradas, saídas e saldo de produtos, além de calcular automaticamente o preço médio e o valor total em estoque.

## 📂 Conteúdo
- `ControleDeEstoque.xlsx` → Planilha principal para controle de inventário.

## 🛠 Funcionalidades
- Registro de entradas e saídas de produtos.
- Cálculo automático do preço médio ponderado.
- Atualização do saldo em quantidade e valor.
- Campos para especificação de produto, fornecedor, cor e tamanho.

## 🚀 Como usar
1. Baixe o arquivo `ControleDeEstoque.xlsx`.
2. Abra no Excel ou software compatível (LibreOffice, Google Sheets).
3. Preencha os campos de **entrada** e **saída** conforme movimentações de estoque.
4. O saldo e o preço médio serão atualizados automaticamente.

## 📌 Observações
- Este modelo pode ser adaptado para diferentes tipos de negócio.
- Recomenda-se salvar versões periódicas para manter histórico.
